My Movies API

Objective:

Build the server side component of a 'movie' web application. It will provide users with access to information about movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

Endpoints

Business Logic URL HTTP Method Request Body Format Response Body Format
Register new user /users/register POST A JSON object with data about the user:
{
"Name": "James Comey",
"Username": "BigMoney20",
"Password": "password",
"Email": "email@domain.com",
"Birthdate": "1999-01-01"
} 		A text message confirming a new account has been created for (username)
Update user info /users/:username PUT {
"Name": "fName lName",
"Username": "newUserName",
"Password": "newPassword",
"Email": "email@domain.com",
"Birthdate": "1999-01-01",
} 		A text message indicating the user info has changed with new user info displayed as a json
Add movie to user's favorites /movies/:MovieID PUT None A text message confirming that (MovieID) has been added to your favorite movies
Remove movie from user's favorites /movies/:MovieID DELETE None A text message confirming (MovieID) has been removed from favorite movies
Delete user account /users/:username DELETE None A text message confirming the account (username) has been deleted
Return a list of all movies /movies GET None A JSON array containing all movie objects
Search movies by title /movies/:Title GET None A JSON object containing a description of the movie, genre, director, image URL and wether it's featured
Search genres by name /genres/:Name GET None A text message containing a description of the genre
Search directors by name /directors/:Name GET None A text message containing a biography with birth and death years
