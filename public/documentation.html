My Movies API

Objective:

Build the server side component of a 'movie' web application. It will provide users with access to information about movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite movies.

Endpoints

Business Logic URL HTTP Method Request Body Format Response Body Format
Return a list of all movies /movies GET None A JSON array containing all movie objects
Return info about a single movie /movies/info/:name GET None A JSON object containing a description of the movie, genre, director, image URL and wether it's featured
Return description of a genre /movies/genres/:name GET None A text message containing a description of the genre
Return data on a director /movies/directors/:name GET None A text message containing a biography with birth and death years
Register new user /users/register POST A JSON object with data about the user:
{
name: "James Comey",
username: "BigMoney20"
password: "password"
} 		A text message confirming a new account has been created for (username)
Allow users to update their username /users/update/:new-name PUT None A text message indicating the username has changed to (new username)
Allow users to add favorite movies to a list /movies/add POST A JSON object with data about the movie:
{
title: `Grandma's Boy`,
director: 'Nicholaus Goosen'
} 		A text message confirming that (title) has been added to your favorite movies
Allow users to remove movies from favorites /movies/remove/:title DELETE None A text message confirming (title) has been removed from favorite movies
Allow users to delete their account /users/deregister/:username DELETE None A text message confirming the account (username) has been deleted
